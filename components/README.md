# web-vue3x

基于 vue3.x 的业务组件库。

[查看组件 demo](https://zhoubichuan.com/web-vue3x/senior/4.component/1.index.html)

## 1.使用方法

### 1.1 安装

以 yarn 为例

```sh
yarn add web-vue3x
```

### 1.2 配置

main.js

```js
import Webvue3x from 'web-vue3x'
...
Vue.use(Webvue3x) // 全局引入
```

### 1.3 使用

- 全局引入可以直接使用，需要先在此组件前面先配置好`vue3x`
